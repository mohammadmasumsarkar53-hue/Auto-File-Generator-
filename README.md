# Telegram File Generator Bot for Railway

## ডেপ্লয় ধাপ
1. GitHub এ নতুন রিপোজিটরি তৈরি করুন
2. সব ফাইল আপলোড করুন
3. Railway.app এ সাইনআপ করুন
4. New Project → Deploy from GitHub → আপনার রিপো নির্বাচন করুন
5. Environment Variables → TELEGRAM_TOKEN = আপনার বট টোকেন দিন
6. Deploy স্বয়ংক্রিয় হবে
7. বট রান হবে (পোলিং মোড)

## ফ্রি টিয়ার সীমা
- ৫০০ ঘন্টা/মাস (~২০ দিন)
- ক্রেডিট কার্ড যুক্ত করলে মাসে ঘন্টা রিসেট হয়
