# Telegram File Generator Bot

এই বটটি ব্যবহার করে আপনি টেলিগ্রামে ফাইলের নাম এবং কন্টেন্ট পাঠিয়ে যেকোনো টেক্সট ফাইল তৈরি করতে পারবেন (যেমন .txt, .py, .html, .js)। বড় কন্টেন্ট একাধিক মেসেজে ভাগ করেও পাঠাতে পারবেন।

## ফিচার
- একাধিক মেসেজে কন্টেন্ট জমা নেয়
- ফাইলের নাম ভ্যালিডেশন (সিকিউরিটি)
- যেকোনো টেক্সট ফাইল জেনারেট করতে পারে
- ফ্রি হোস্টিং Render + UptimeRobot এর জন্য তৈরি

## Render এ ডিপ্লয়
1. এই রিপোজিটরি ফর্ক করুন।
2. Render এ নতুন Web Service তৈরি করুন।
3. গিটহাব রিপো সংযোগ করুন।
4. এনভায়রনমেন্ট ভেরিয়েবল সেট করুন: `TELEGRAM_TOKEN` = আপনার বট টোকেন
5. Build Command: `pip install -r requirements.txt`
6. Start Command: `python app.py`

## UptimeRobot দিয়ে সক্রিয় রাখুন
Render ফ্রি সার্ভিস ১৫ মিনিট পর ঘুমিয়ে যায়। তাই UptimeRobot এ একটি মনিটর তৈরি করুন:
- Type: HTTP(s)
- URL: `https://আপনার-অ্যাপ.onrender.com/health`
- Interval: ৫ মিনিট

## বটের কমান্ড
- `/start` - স্বাগতম বার্তা
- `/create` - ফাইল তৈরির প্রক্রিয়া শুরু
- `/done` - কন্টেন্ট পাঠানো শেষ হলে
- `/cancel` - প্রক্রিয়া বাতিল
